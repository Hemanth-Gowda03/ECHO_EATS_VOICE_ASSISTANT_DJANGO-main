# Food Booking Web Application with Voice Assistance

A Django-based web application that enables users to order food using voice commands. The system integrates speech-to-text processing and intent recognition to provide a seamless and accessible food ordering experience.

## Features

* Voice-based food ordering
* Speech-to-text integration
* Intent recognition and command processing
* Menu browsing and food selection
* Order management
* Responsive user interface

## Technologies Used

* Python
* Django
* SQLite
* HTML
* CSS
* JavaScript
* APIs

## Installation

1. Clone the repository
2. Install dependencies
3. Run migrations
4. Start the development server

```bash
git clone <repository-url>
cd FoodBookingApp
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```
